# Scene 1

## ヒント

### ヒント1

「右矢印キー」5回では、クリアすることができません。

本ゲームは「マスモード」の表示をもとに、「入力モード」でそれに対応する適切な入力を行うことでクリアすることができます。

注: 
- マスモード: マスへフォーカスが当たっているときのモード
- 入力モード: 矢印の入力欄にフォーカスが当たっているときのモード

### ヒント2

太い枠線はその方向に黒マスが隣接していることを示し、枠線がない場合、その方向に白マスが隣接していることを示します。

### ヒント3

マスからマスへどのように移動したか、考えてみましょう。

## 解答

<details>

<summary>ここをクリック</summary>

本作は、マスの表示からそれに対応する移動を推測するゲームです。

マスの左上に表示されている小さな数字を以下「インデックス」と呼ぶことにします。インデックスはSからGに到達するまでの移動回数です。

例えば、インデックス0から1への移動について考えてみましょう。インデックス0のマスは、右が白マスであり、インデックス1のマスは左と右が白マスです。したがって、インデックス0と1のマスは左右に隣接しており、インデックス0から1へ右方向に移動していると考えられます。

他のマスについても同様に考えてみると、以下のように移動が決定されます。

解答例: `→2↓1→2`
